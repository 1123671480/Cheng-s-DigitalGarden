地理空间范围
KML(Keyhole Markup Language,Keyhole 标记语言)最初是由 Google 旗下的 Keyhole 公司开发和维护的一种基于 XML 的标记语言，利用 XML 语法格式描述地理空间数据(如点、线、面、多边形和模型等)，适合网络环境下的地理信息协作与共享。2008 年4月,KML 的最新版本2.2 被 OGC 宣布为开放地理信息编码标准,并改由 OGC 维护和发展。
