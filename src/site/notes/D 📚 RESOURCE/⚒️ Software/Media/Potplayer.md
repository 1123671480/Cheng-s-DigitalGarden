# 快捷键
==TAB== 显示视频 OSD 信息
==D== ==F== 上下帧
==Ctrl== + ==Shift== + ==←→== 关键帧移动
==C== ==X== 加减速播放
==Z== 倍速播放切换
==[ ]== 标记区间，可循环播放
=={ }== 取消区间标记
==P== 添加书签
==H== 打开书签界面
==K== 截图格式
==Alt== + ==C== 录制视频
